# 🚀 DataFlow-PHP

A simple, secure PHP & MySQL application that collects user data (name and age) via an HTML form and saves it safely to a database. ✨

---

## 📁 File Structure

*   **`index.html`** 📝 — User registration form (Frontend).
*   **`in.php`** ⚙️ — Processes and saves form data to the database safely using **PDO**.
*   **`database.sql`** 🗄️ — SQL schema to set up the database table.
*   **`README.md`** 📖 — Project documentation.

---

## 🛠️ Setup & Installation

1. **Database Setup 🗃️:** Import `database.sql` via phpMyAdmin to create the `user` table.
2. **Configuration 🔑:** Update the database credentials inside `in.php` to match your server.
3. **Deploy 🚀:** Upload all files directly to your server's root directory (e.g., `htdocs` on **InfinityFree** or **XAMPP**).
4. **Run 🎉:** Navigate to your domain in the web browser to start using the app!
